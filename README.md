A tiny, adorable Droid-style voice pack.

A small droid-style voice pack with higher pitched sounds inspired by cute robots like WALL-E.
Just a fun little project to make devices sound more like a tiny robot.


This repository contains a modified voice pack based on: 
- https://github.com/Makers-Im-Zigerschlitz/voicepacks_dreame/tree/main/soundpacks (R2D2 Sound)
- https://github.com/czaky/dreame_voice_pack (Glados)

The original sounds were pitched up to a higher frequency so they sound a bit more like a small, cute robot.
Some additional randomly sampled droid sounds were also added.

All audio editing was done with Audacity:
- use Audacity macro "Change pitch", set to 370Hz
- export as .ogg

## How to use
Open Valetudo Web UI

Go to Robot -> System Options (to the bottom) -> Voice packs

Add the URL: https://github.com/jpalbrecht/wall-e-tudo/blob/main/voice_walle.tar.gz (if your robot is disconnected from internet, you have to set up an own web server, you ll know how to do that because you re a nerd.)

Add language code (do not use EN)

Add MD5 for voice_walle.tar.gz:
bacccd175afd099e342412422ac63ae8
